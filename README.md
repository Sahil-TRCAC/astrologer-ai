# 🔮 Celestial Astrologer AI

An AI-powered astrology web application that provides personalized cosmic readings using Google's Gemini AI.

## ✨ Features

- **AI-Powered Readings**: Get personalized astrology insights using Google Gemini AI
- **Zodiac Personalization**: Select your zodiac sign for tailored readings
- **Birth Date Integration**: Age-aware responses for better guidance
- **Rich UI Experience**:
  - Animated star background
  - Glass morphism design
  - Floating planets effect
  - Responsive layout for all devices
- **Reading Management**:
  - Copy to clipboard
  - Share readings
  - Download as text file
  - Text-to-speech audio
- **Cosmic Sound Effects**: Optional ambient sounds (432Hz frequency)
- **Auto-save**: Form data persists across sessions
